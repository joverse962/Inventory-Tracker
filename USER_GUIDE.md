## Using the Dashboard

### Dashboard Overview
The main dashboard shows:
- **Total Items**: All items in inventory
- **Available**: Items ready to use
- **Borrowed**: Items currently checked out
- **In Use**: Items actively being used

### Viewing Items
- Items are displayed as cards with:
  - Photo (if uploaded)
  - Item name
  - Status (Available, Borrowed, In Use, Waste)
  - Location
  - Category
  - Barcode

### Item Details
Click any item card to see full details:
- Complete description
- Components/accessories
- Current user (if borrowed)
- Creation date
- Action buttons (Borrow/Return)

## Adding New Items

### Step-by-Step:

1. **Click "Add Item"** button (at the top of dashboard)

2. **Fill in Basic Information:**
   - **Item Name** *(required)*: e.g., "Dell Laptop XPS 15"
   - **Description** *(required)*: Detailed info about the item
   - **Category** *(required)*: e.g., Electronics, Furniture, Tools
   - **Location** *(required)*: e.g., Warehouse A, Office Floor 2

3. **Set Quantity:**
   - Default is 1
   - Increase for multiple identical items

4. **Upload Photo:**
   - Click "Choose File"
   - Select an image (JPEG, PNG, or WebP)
   - Maximum file size: 5MB
   - Preview will show automatically

5. **Add Components** (optional):
   - Click "+ Add Component"
   - Enter component name and quantity
   - Example: "Power Adapter - 1", "Mouse - 1"
   - Click ‚úï to remove components

6. **Barcode** (optional):
   - Leave empty to auto-generate
   - Or enter existing barcode number

7. **Click "Add Item"** button

8. You'll be redirected to the dashboard where your new item appears!

## Borrowing Items

### To Borrow an Item:
1. Find the item on dashboard
2. Click the item card to open details
3. Verify status is **"Available"** (green)
4. Click **"Borrow"** button
5. Confirmation message appears
6. Item status changes to **"Borrowed"** (yellow)
7. Your name appears as "Current User"

### Rules:
- Only **Available** items can be borrowed
- One user can borrow multiple items
- Items must be returned before others can borrow them

---

## Returning Items

### To Return a Borrowed Item:
1. Go to dashboard or your **Profile** page
2. Find the borrowed item
3. Click the item card
4. Click **"Return"** button
5. Confirmation message appears
6. Item status changes back to **"Available"**
7. Item can now be borrowed by others

### Your Profile:
View all items you currently have borrowed:
1. Click **"Profile"** in navigation
2. See list of your borrowed items
3. Click **"Return"** on any item

## Searching & Filtering

### Search:
- Use the **search box** at the top
- Searches in: Name, Description, Barcode
- Press Enter or click search icon

### Filters:
- **Status**: All, Available, Borrowed, In Use, Waste
- **Category**: Type to filter (e.g., "Electronics")
- **Location**: Type to filter (e.g., "Warehouse A")

### Clear Filters:
- Remove text from filter boxes
- Select "All Statuses" in status dropdown

## Common Questions

**Q: Can I edit an item after adding it?**
A: Currently items can be updated by clicking them and using edit options (if your role allows).

**Q: What if I forgot to add a photo?**
A: You can edit the item and upload a photo later.

**Q: Can I borrow multiple items?**
A: Yes! Borrow as many as you need.

**Q: How do I see what I've borrowed?**
A: Go to your Profile page to see all your borrowed items.

**Q: What's the difference between "Borrowed" and "In Use"?**
A: "Borrowed" means checked out by a specific user. "In Use" means actively being used but not assigned to anyone.

**Q: Can I delete items?**
A: Admins can delete items. Regular users can mark items as "Waste".

**Q: Is there a mobile app?**
A: The website works on mobile browsers! Just bookmark it on your phone.

**Q: How do I log out?**
A: Click the **"Logout"** button at the top right of the dashboard.
